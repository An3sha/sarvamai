🤖 Sarvam AI Agent Widget

🚀 Quick Start

Add the Sarvam AI Agent Widget to your website with just a few lines of code:

<!-- Include React and ReactDOM --> <script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script> <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script> <!-- Include the Sarvam Agent Widget --> <script src="https://your-cdn.com/sarvam-agent-widget.umd.js"></script> <!-- Configure the widget --> <script> window.AgentWidgetConfig = { sarvamApiKey: 'your-api-key-here', position: 'bottom-right', theme: { primaryColor: '#4F46E5', background: '#ffffff', text: '#111827' }, agent: { name: 'HelperBot', avatar: 'https://example.com/avatar.png', greeting: 'Hello! How can I help you today?' }, enableVoice: true, languages: ['en', 'hi', 'ta', 'te', 'bn'], // Popular Indian languages enableAutoTranslation: true, // Auto-translate when switching languages translationMode: 'formal', // Use formal translation mode enablePreprocessing: true, // Better translations for complex text context: 'You are a helpful AI assistant for our website.' }; </script>

✨ Features

  • Multi-language support (22 Indian languages: English, Hindi, Tamil, Telugu, Bengali, and more)
  • Automatic translation when switching languages
  • Advanced translation modes (formal, colloquial, code-mixed)
  • Voice input and output capabilities
  • Customizable appearance and positioning
  • Context-aware AI responses
  • Mobile-responsive design
  • Shadow DOM isolation (no CSS conflicts)
  • Easy integration with any website

⚙️ Configuration Options

Basic Configuration:

window.AgentWidgetConfig = { // Required: Your Sarvam API key sarvamApiKey: 'your-api-key-here', // Optional: Widget position position: 'bottom-right', // 'bottom-left', 'top-right', 'top-left' // Optional: Theme customization theme: { primaryColor: '#4F46E5', background: '#ffffff', text: '#111827', font: 'Inter, system-ui, sans-serif' }, // Optional: Agent customization agent: { name: 'HelperBot', avatar: 'https://example.com/avatar.png', greeting: 'Hello! How can I help you today?' }, // Optional: Features enableVoice: true, autoOpen: false, showWelcomeMessage: true, // Optional: Supported languages languages: ['en', 'hi', 'es'], // Optional: AI context context: 'You are a helpful AI assistant for our website.', // Optional: UI customization placeholder: 'Type your message...', maxMessages: 50 };

🎤 Voice Features

The widget includes advanced voice capabilities:

  • Speech-to-text using browser APIs or Sarvam STT
  • Text-to-speech using browser APIs or Sarvam TTS
  • Multi-language voice recognition
  • Automatic voice response in voice mode
  • Fallback to browser APIs when Sarvam APIs are unavailable

🔧 Advanced Usage

Manual Widget Mounting:

// Mount widget manually window.SarvamAgentMount(); // Or with custom configuration window.AgentWidgetConfig = { /* your config */ }; window.SarvamAgentMount();

Dynamic Configuration:

// Update configuration dynamically window.AgentWidgetConfig = { ...window.AgentWidgetConfig, theme: { primaryColor: '#10b981' // Change to green } };
